# Tello Drone Python Controller

小型ドローン「Tello」をPythonで自動制御するプログラミングです。
文系大学生として、プログラミング学習の一環で「実際にモノを動かす楽しさ」を体験するために作成しました。

## 概要
PCとTelloをWi-Fiで接続して操作します。
「離陸 → 指定のアクション → 着陸」という一連の飛行プロセスを自動で完結させることを目指しました。

## 実装した主な機能
- **離着陸の自動実行**: ボタン操作なしで安全に離陸・着陸を行います。
- **移動制御**: 前後左右、上下への移動をプログラムで指示します。

## 使用ツール
- **言語**: Python
- **エディタ**: Visual Studio Code
- **使用機体**: Tello
